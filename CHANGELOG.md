# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](http://keepachangelog.com/en/1.0.0/)
and this project adheres to [Semantic Versioning](http://semver.org/spec/v2.0.0.html).

## [1.2.0] - 2026-02-16

### Added

- Added an `AutoPageTurn` property to enable/disable continuous page turning.
- Added a custom UI Toolkit inspector for `StrobePagesController` with a manual page-turn button.

## [1.1.0] - 2026-02-12

### Added

- Added a Pass Event property to `StrobePagesRendererFeature` to control the render pass injection point.

### Changed

- Added a URP package dependency (`com.unity.render-pipelines.universal` 17.0.0) to `package.json`.
- Reordered `StrobePagesController` properties.

## [1.0.0] - 2026-02-09

### Added

- Initial release.
